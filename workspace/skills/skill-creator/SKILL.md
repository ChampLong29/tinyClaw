# 技能创建器

当你使用 `/创建技能` 命令时，按以下步骤帮助用户创建新技能：

## 创建流程

1. **询问技能信息**
   - 技能名称（中文）
   - 触发指令（以 / 开头的中文命令）
   - 技能描述（一句话说明功能）
   - 技能详细说明（具体功能和使用方法）

2. **生成技能文件**
   在 `workspace/skills/<技能目录名>/SKILL.md` 下创建文件：

   ```markdown
   ---
   name: <技能名称>
   description: <技能描述>
   invocation: <触发指令>
   ---
   # <技能名称>

   <技能详细说明>
   ```

3. **确认创建**
   告诉用户技能已创建，并说明如何使用 `/<触发指令>` 来调用。

## 示例对话

用户：`/创建技能`
助手：好的！我来帮你创建一个新技能。请告诉我：
- 技能名称是什么？
- 触发指令是什么？（以 / 开头）
- 技能描述是什么？
- 技能的具体功能是什么？

用户：`名称=天气查询，触发指令=/天气，描述=查询天气预报，功能=当用户说/天气时，告诉用户当前城市并说明如何使用天气查询功能`

助手：已为你创建「天气查询」技能！文件位置：`workspace/skills/天气查询/SKILL.md`

使用 `/天气` 即可调用此技能。

## 注意事项

- 技能目录名使用中文
- 触发指令必须以 / 开头
- 技能内容使用 Markdown 格式
- 创建后技能会在下次启动时自动加载
